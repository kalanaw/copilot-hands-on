# React / TS Frontend boilerplate


### Features
- [x] Typescript
- [x] React
- [x] Sass

### How to run

cd into the frontend folder and run the following commands:

- `npm install`
- `npm start` - run the server at [http://localhost:3000](http://localhost:3000)
